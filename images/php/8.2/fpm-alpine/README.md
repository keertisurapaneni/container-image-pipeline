## PHP FPM 8.2

This image is a fully functional all-in-one php stack that includes:

- PHP 8.2
- PHP FPM
- Nginx
- Supervisor
- New Relic Agent

Each component has sane defaults where applicable and allows for overriding in your own projects as needed.
